Introdução

Um programa é uma sequência de instruções que pode ser interpretada e executada pelo computador para resolver um determinado problema. Em C, a execução do programa começa na função main(), e todo o código pertencente a essa função deve ser colocado entre { }, formando um bloco. 

 
main() {

}

Todo e qualquer programa, se inicia pela função main. Toda e qualquer função, deve ser seguida por `()` se eles não tem nada nesses parênteses, quer dizer que não receber nada exterior.

Não esqueça que C é case sensitive, diferenciando minúsculas e maiúsculas.

Hello World

 
#include 
main() {
	printf("Hello World");
}

1. Em C, sempre que quisermos tratar caracteres, devemos colocá-los entre "", não troque por ''

2. Uma das funções em C que permite a escrita na tela é a printf, sendo print + formatação. Como ela se trata de uma função, deve ter parênteses, nesse caso, passamos uma String de caracteres. O uso dessa função é permitido pela biblioteca stdio.h

3. Em C, cada instrução deve ser termina por ;

4. Arquivos que não tem código, apenas cabeçalhos de funções, são denominados header files e são seguidos de uma extensão .h

O mesmo programa também poderia ser escrito de diferentes formas: 

 
// Forma 01
#include 
main () {
	printf("Hello");
	printf(" ");
	printf("World"); 
}

// Forma 02
#include 
main () {
	printf (
		"Hello"
	);
	printf (   " " )
	;
	printf ("World");}

Como pular uma linha

Para isso, utilizamos a sequência de escape New Line, representada por \n. Isso acontece porque a linguagem C não possui uma função específica para imprimir uma nova linha, como ocorre em algumas outras linguagens. 

 
#include 
main(){
	printf("Hello World\n");
}
O \n pode ser utilizado em qualquer ponto da string, assim como qualquer outro caractere.

Como exibir aspas duplas

Para exibir aspas duplas dentro de uma string, colocamos uma barra invertida (\) antes delas. Dessa forma, elas deixam de ser interpretadas como delimitadores da string. 

 
#include 
int main() {
	printf("Hoje está um \"Lindo\" Dia\n");
	return 0;
}

O Caractere \

A barra invertida (\) é utilizada para criar sequências de escape, removendo o significado especial de determinados caracteres ou representando caracteres que não podem ser digitados diretamente.

Sequência Significado
\7 Bell: sinal sonoro do computador
\a Bell: sinal sonoro do computador
\b Backspace
\n Nova linha (line feed)
\r Retorno de carro (carriage return)
\t Tabulação horizontal
\v Tabulação vertical
\\ Caractere \ literal
\' Aspas simples (')
\" Aspas duplas (")
\? Caractere ?
\ooo Caractere cujo código ASCII em octal é ooo
\xnn Caractere cujo código ASCII em hexadecimal é nn
\%%

Comentários

Comentários são ignorados pelo compilador e servem apenas para documentar o código-fonte.
Em C, um comentário tradicional é qualquer sequência de caracteres delimitada por /* e */.

Exemplo de Cabeçalho

 
/* PROG0199.C :     Comentários e Companhia    */
/* AUTOR      :     Luís Damas                 */
/* DATA       :     01/11/1995                 */

Comentários também podem ser inseridos dentro de expressões. Entretanto, essa prática geralmente reduz a legibilidade do código. 

 
printf("Olá mundo" /* Falta o \n */);

Existe apenas uma situação em que a sequência /* */ não é interpretada como comentário: quando ela aparece dentro de uma string. Nesse caso, ela perde seu significado especial. 

 
printf("Um comentário em C começa com /* e termina com */");

Alguns compiladores oferecem suporte a comentários aninhados (nested comments), mas essa funcionalidade não faz parte do padrão tradicional da linguagem C.

Resumo

Um programa em C inicia sua execução pela função main(), e todo o código pertencente a essa função deve ficar entre { }. Esse conjunto de instruções forma um bloco, e cada instrução deve terminar com um ponto e vírgula (;).

A linguagem é sensível a maiúsculas e minúsculas (case-sensitive), e strings são escritas entre aspas duplas (").

Para exibir informações na tela, utiliza-se a função printf(), disponibilizada pela biblioteca padrão stdio.h. Por isso, é necessário incluí-la por meio da diretiva #include.

As diretivas iniciadas por # pertencem ao pré-processador e, diferentemente das instruções da linguagem, não terminam com ponto e vírgula.

Caracteres especiais são representados por sequências de escape, iniciadas pela barra invertida (\). Já os comentários são delimitados por /* e */ e, em geral, não podem ser aninhados.
